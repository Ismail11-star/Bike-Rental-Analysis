
🚲 Bike Rental Demand Analysis Dashboard
📌 Project Overview
This project features an interactive Power BI Dashboard designed to analyze and visualize bike rental trends. By processing raw data from the BikeDataforRental.csv dataset, the dashboard identifies key factors influencing rental demand, such as seasonal changes, weather conditions, and user demographics.

This project demonstrates core competencies in Data Visualization, Business Intelligence (BI), and Exploratory Data Analysis (EDA).

📊 Key Features & Insights
Seasonal Demand Tracking: Visualizes peak rental periods across Spring, Summer, Fall, and Winter.

Weather Impact Analysis: Correlates rental frequency with weather conditions (Clear, Mist, Light Snow/Rain).

User Segmentation: Distinguishes behavior between casual riders and registered members.

Interactive Slicers: Allows users to filter the entire dataset by specific timeframes or weather types for granular insights.

KPI Metrics: Real-time calculation of Total Rentals, Average Humidity, and Peak Usage Hours.
